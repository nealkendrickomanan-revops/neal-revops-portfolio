## 🔧 Case Study: CSV Cleanup → HubSpot-Ready Data

### Problem
480-row contact dataset with duplicate emails, inconsistent name casing, mixed date formats, invalid emails, and unstandardised lifecycle stages — not ready for HubSpot import.

### What I did
- Removed 179 duplicate contacts (email-based deduplication)
- Standardised name casing, phone format, and date format (YYYY-MM-DD)
- Normalised lifecycle stage values to HubSpot internal format
- Validated and removed invalid emails before import

### Result
480 rows → 165 clean contacts successfully imported into HubSpot with 0 mapping errors.
